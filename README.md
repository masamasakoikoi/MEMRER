# MENRER（メンラー）

## サイト概要

### サイトテーマ
お気に入りのラーメン屋をシェアできるSNSサイトです

### テーマを選んだ理由
ラーメンを食べるのが好きで、自分が気に入ったラーメン屋を投稿してシェアできるサービスがあったら良いと思い開発しました。

### ターゲットユーザ
- 気に入ったラーメン屋をシェアしたい人
- ラーメン屋のリアルな口コミを見たい人

### 主な利用シーン
- ラーメン屋に行きたくなったとき
- 口コミを見てからどこのラーメン屋に行くか判断したい人
